# Sailing Assistant

B&G NMEA 2000 alapú vitorlás tour-race asszisztens app. A YDWG-02 WiFi gateway-ről
érkező adatok alapján real-time számolja a következő bója utáni TWA-t,
bearing-to-mark, course correction, ETA és wind shift trend értékeket.

Telefon (Pixel) + Wear OS óra (Samsung) szinkronban — a versenyző az órájára
pillantva minden lényeges navigációs adatot megkap, kéz nélkül.
